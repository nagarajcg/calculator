🧮 Union Style Scientific Calculator

A modern scientific calculator built using HTML, CSS, and JavaScript with a clean glassy gradient background and union-style design.
It supports all basic and advanced scientific operations like addition, subtraction, multiplication, division, power, logarithmic, and trigonometric functions.

🚀 Features

✅ Modern glassmorphic union-style UI
✅ Works offline — no external libraries
✅ Supports:

Basic operations: +, -, ×, ÷, %

Power x^y

Trigonometric: sin, cos, tan

Logarithmic: ln, log

Square root √

Constants: π, e

Factorial x!

Memory functions: MC, MR, M+, M-
✅ Keyboard input support
✅ Responsive and fast performance
